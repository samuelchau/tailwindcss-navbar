npx create-react-app tailwindcss-navbar
cd tailwindcss-navbar


npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

In tailwind.config.js, add the following:
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

In index.css, add the lines:
@tailwind base;
@tailwind components;
@tailwind utilities;

Run 'npm run start'
